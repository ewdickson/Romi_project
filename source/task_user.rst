User Task
=========
The user task provides a lightweight serial user interface for interacting with the 
robot during operation. It reports startup messages, indicates when the robot begins 
or stops running, and prints finite-state-machine state changes as they occur. In 
this project, it mainly serves as a debugging and monitoring tool, making it easier 
to observe the robot’s behavior without modifying the lower-level control logic.
