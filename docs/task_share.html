class task_share.BaseShare(type_code, thread_protect=True, name=None)

Bases: object

Base class for queues and shares used to exchange data between tasks.

This class stores configuration shared by Queue and Share objects and registers each instance in the global diagnostic list.

Queue used to transfer buffered data from one task to another.

A Queue stores multiple values of one data type and can optionally protect data transfers from corruption caused by interrupts.
