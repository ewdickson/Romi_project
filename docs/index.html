Project Summary

This project is an autonomous line-following robot designed for the ME405 term project. Depending on track position, the robot detects and follows printed lines, naviagates predetermined paths, or recovers from bumping a wall.

General Strategy

Our team designed and built a robot capable (maybe) of completing repeated time trials on the printed lab track primarily using line sensing and encoder feedback in combination with PI motor controllers.
